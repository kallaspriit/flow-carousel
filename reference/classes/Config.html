getRenderRange

(
  • currentItemIndex
  • itemsPerPage
  • itemCount
  • firstRender
)
Object private

Defined in src\Config.js:405

Returns the range of items that should be rendered given current item index and items per page.

By default returns one page before the current page and one after but one may choose to override it. On the first render, returns three pages before/after so that the user can navigate a few pages without triggering rendering new items.

One can use the firstRender parameter to for example render more items during the first render so no new rendering is triggered if the user only moves for a few pages.

Parameters:

  • currentItemIndex Number

    Currently scrolled position index

  • itemsPerPage Number

    How many items are shown on a page

  • itemCount Number

    How many items there are in total

  • firstRender Boolean

    This parameter is true for the very first render

Returns:

Object:

Render range with keys "start" and "end"

cssClasses

Object

Defined in src\Config.js:313

CSS class names to use.

The class name is combined with the cssPrefix property so if the prefix is "flow-carousel-" and the wrap class is "wrap" then the main wrap will get the "flow-carousel-wrap" class.

Sub-properties:

  • wrap='wrap' String

    Assigned to the main wrap element

  • items='items' String

    Assigned to the items wrap element in the main wrap

  • scroller='scroller' String

    Assigned to the animated scroller wrap in the items wrap

  • item='item' String

    Assigned to each item wrapper containing the actual item

  • itemHover='item-hover' String

    Assigned to item wrapper on hover and removed on mouse out

  • placeholder='placeholder' String

    Assigned to each item wrapper that is a placeholder

  • matchWrap='match-wrap' String

    Assigned to main wrap when using the wrap size match mode

  • matchLargestItem='match-largest-item' String

    Assigned to main wrap when matching the wrap size to the largest item size

  • horizontal='horizontal' String

    Assigned to main wrap for horizontal orientation

  • vertical='vertical' String

    Assigned to main wrap for vertical orientation

  • animateTransform='animate-transform' String

    With this class added the transforms get animated as well

  • initiating='initiating' String

    Assigned to main wrap during initialization procedure

  • loading='loading' String

    Assigned to main wrap during loading of items

  • rendering='rendering' String

    Assigned to main wrap during rendering of items

responsiveLayoutDelay

Number

Defined in src\Config.js:101

How long to wait for the wrap size to stay the same before starting the responsive layout routine.

This is useful so when the user is resizing the browser window then the carousel won't try to re-layout itself on every frame but rather waits for the size to stabilize.
