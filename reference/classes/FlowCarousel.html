Event

Object

Defined in src\FlowCarousel.js:513

List of possible events emitted by the carousel.

The event system uses the Wolfy87 EventEmitter implementation https://github.com/Wolfy87/EventEmitter.

You can subscribe to events using carousel.addListener(carousel.Event.INITIATED, function() { ... }); etc

The [startIndex, endIndex] syntax shows the parameters passed to the callback.

Sub-properties:

  • INITIATING='initiating' String

    Emitted when starting initiation

  • INITIATED='initiated' String

    Emitted after completing initiation

  • STARTUP_ITEMS_RENDERED='startup-items-rendered' String

    Emitted after rendering the first data

  • LOADING_ITEMS='loading-items' String

    [startIndex, endIndex, items] Emitted when starting to load a new set of items

  • LOADED_ITEMS='loaded-items' String

    [startIndex, endIndex, items] Emitted when a new set of items was loaded

  • ABORTED_ITEMS='aborted-items' String

    [startIndex, endIndex] Emitted when loading a range of items was aborted

  • RENDERED_ITEMS='rendered-items' String

    [startIndex, endIndex, elements] Emitted when a new set of items was rendered

  • DESTROYED_ITEMS='destroyed-items' String

    [itemIndexes] Emitted when a set of items was destroyed

  • NAVIGATING_TO_ITEM='navigating-to-item' String

    [itemIndex, instant] Emitted when starting to navigate to a carousel item index

  • NAVIGATED_TO_ITEM='navigated-to-item' String

    [itemIndex, instant] Emitted when finished navigate to a carousel item index including animation

  • NAVIGATING_TO_PAGE='navigating-to-page' String

    [pageIndex, instant] Emitted when starting to navigate to a carousel item index

  • NAVIGATED_TO_PAGE='navigated-to-page' String

    [pageIndex, instant] Emitted when finished navigate to a carousel item index including animation

  • LAYOUT_CHANGED='layout-changed' String

    Emitted when the layout is re-calculated

  • DRAG_BEGIN='drag-begin' String

    [startPosition, dragOppositePosition, carouselPosition] Emitted when the drag navigator begins to drag the scroller

  • DRAG_END='drag-end' String

    [navigationMode, startPosition, endPosition, deltaDragPosition, closestIndex, direction, targetElement] Emitted when dragging stopped
