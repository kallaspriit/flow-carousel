SizeMode

Object static

Defined in src\FlowCarousel.js:229

There are two different strategies for setting the size of the wrap and the items:

MATCH_WRAP - the size of the items is set to match the wrap size MATCH_LARGEST_ITEM - the size of the wrap is set to match the largest item

Shortcut to the list of possible size modes from Config.

Sub-properties:

  • Orientation.MATCH_WRAP='match-wrap' String

    Items size is based on wrap size

  • Orientation.MATCH_LARGEST_ITEM='match-largest-item' String

    Wrap size is based on items size
