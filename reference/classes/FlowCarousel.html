FlowCarousel chainable

Defined in src\FlowCarousel.js:133

Sets the data source to use.

Expects either an instance of AbstractDataSource (you can roll your own) or a simple array that will be converted to an ArrayDataSource.

This can optionally be set in the init method as second argument.

This method supports call chaining by returning itself.
