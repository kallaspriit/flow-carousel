)
Deferred.Promise

Inherited from AbstractAnimator but overwritten in src\TransformAnimator.js:172

Animates the carousel to given absolute position in pixels.

One can set either a custom animation speed in pixels per millisecond or custom animation duration in milliseconds. If animation duration is set then animation speed is ignored.

Animation speed from Config/transformAnimator/defaultAnimationSpeed:property is used by default.

Returns deferred promise that is resolved when the animation completes.

Parameters:

  • position Number

    Requested position in pixels

  • [instant=false] Boolean optional

    Should the navigation be instantaneous and not use animation

  • [noDeferred=false] Boolean optional

    Does not create a deferred if set to true, returns undefined

  • [animationSpeed] Number optional

    Animation speed in pixels per millisecond

  • [animationDuration] Number optional

    Optional animation duration in milliseconds
